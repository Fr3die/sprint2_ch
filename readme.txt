*Para voce iniciar o codigo, voce precisa executar os seguintes comandos

*Processo para Linux

1 - apt install pip
2 - acesse o diretorio onde está o codigo e seu arquivo
3 - execute o comando pip install -r config-start-linux.txt

Cocluido, apenas execute o codigo de geolocalização

*Processo para Windowns
1 - acesse o site (https://www.python.org/) e realize o download da ultima versão do python.
2 - abra o seu power shell executando como admistrador.
3 - Acesse a pasta onde está o arquivo das bibliotecas "config-start-win.txt"
4 - Execute o comando pip install -r config-start-win.txt

Cocluido, apenas execute o codigo de geolocalização 

O desenvolvimento dese codigo foi realizado para o projeto da challenge IBM em 2023 da faculdade FIAP,a intenção do mesmo é registrar e indentificar os containers  inteligentes (TYBA.I.O.T) e identificalos ultilizando bibliotecas de geolocalização.
